****************************************
    Bot Enterprise Framework (MOCHA)
****************************************
Description:
This framework serves as a Enterprise Framework for developing bots using Automation Anywhere v11.x. It contains EnterpriseLogging metabot which is used to Log Bot data to control room database. This data later on use for generating various MIS reports for Bots.

How to use:
     1. Before using this framework user need run database script provided in .\Database 	Script folder. this script will create neccessary Tables, Procedures and Views in 	control room database.
     2. Task templates are provided in .\TemplateTasks folder. One has to use these 	templates for implementing their tasks.
